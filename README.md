# Brief Introduction to Differentiable Programming

## Environment setup
Create a new Python 3.12 environment or use an existing one, then run:

```pip3 install -r requirements.txt```

## Getting started

Start with the [presentation](/diffprog_intro.pdf), take a look at the [example](/example.ipynb), then try your hand at the exercises: [1](/exercise-1.ipynb), [2](/exercise-2.ipynb), [3](/exercise-3.ipynb).

Enjoy!
